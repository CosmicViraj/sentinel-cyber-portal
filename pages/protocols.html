SECURITY PROTOCOLS

🔴
CRITICAL INCIDENT
LEVEL 3
  • Immediately isolate affected systems from network
  • Notify Cyber Command and senior officer on duty
  • Preserve all logs and forensic evidence
  • Activate incident response team within 15 minutes
  • Submit INC report via SENTINEL portal
🟡
DATA BREACH
LEVEL 2
  • Identify and contain the breach source immediately
  • Assess scope — what data was accessed/exfiltrated
  • Revoke all potentially compromised credentials
  • Notify Data Protection Officer within 1 hour
  • Document timeline and affected systems
🟢
PHISHING RESPONSE
LEVEL 1
  • Do not click any links or download attachments
  • Forward the email to cyber@sentinel.mod.gov
  • If credentials entered — change password immediately
  • Report via SENTINEL portal with email headers
  • Warn colleagues if organisation-wide campaign suspected
🔵
RANSOMWARE
LEVEL 3
  • Disconnect affected device from network immediately
  • Do NOT pay ransom — escalate to Cyber Command
  • Identify ransomware strain via SENTINEL AI
  • Restore from last verified clean backup
  • Conduct full forensic analysis post-recovery
